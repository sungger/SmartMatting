# SmartMatting 通用抠图技术文档

## 概述

SmartMatting 的「通用」抠图模式采用**色差法 + 距离变换填孔**方案，不依赖深度学习模型，纯图像处理实现，适用于主体与背景颜色有一定差异的场景。

---

## 算法流程

```
输入图片
  │
  ├─ 1. ensureReadableCGImage ── 确保像素数据可读（RGBA8 格式）
  │
  ├─ 2. sampleEdgeColorFast ── 四角+边缘采样 → 背景色
  ├─ 3. sampleRegionColor ── 中心区域采样 → 前景色
  │
  ├─ 4. 计算色差 → 自适应阈值
  │     colorDiff < 0.10 → threshold = 0.15
  │     colorDiff < 0.20 → threshold = 0.10
  │     colorDiff < 0.35 → threshold = 0.07
  │     默认             → threshold = 0.05
  │
  ├─ 5. pixelMaskFast ── vDSP 向量化生成二值遮罩
  │     远离背景色 → 前景(255)
  │     接近背景色 → 背景(0)
  │
  ├─ 6. 形态学处理
  │     闭运算: 膨胀 5px → 腐蚀 4px（填小孔）
  │     开运算: 腐蚀 2px → 膨胀 1.5px（去噪）
  │
  ├─ 7. binarizeMask ── 极高对比度(20x)二值化
  │     消除形态学操作产生的灰色过渡像素
  │
  ├─ 8. fillInteriorHoles ── 距离变换填孔
  │     计算每个背景像素到最近前景像素的距离
  │     距离 > 1px → 填为前景（内部穿孔）
  │     距离 ≤ 1px → 保留（边缘区域）
  │
  ├─ 9. featherMask ── 0.5px 高斯模糊羽化
  │
  └─ 10. CIBlendWithMask ── 输出带透明通道的抠图结果
```

## 关键方法说明

### ensureReadableCGImage
- **问题**：PhotosPicker 加载的图片 CGImage 像素数据不可直接读取
- **方案**：通过 CGContext 重绘为 RGBA8 格式

### sampleEdgeColorFast
- 四角各采样 2 遍（8 个采样点），取平均值
- 假设主体在中央，四角为背景

### pixelMaskFast
- 使用 vDSP 向量化计算，比逐像素 for 循环快 ~10x
- 计算每个像素与背景色的欧氏距离，与阈值比较

### binarizeMask
- 使用 CIColorControls 对比度 20.0
- 将所有灰色像素强制变为纯黑(0)或纯白(255)
- 消除形态学操作产生的半透明边缘

### fillInteriorHoles (距离变换)
- BFS 从所有前景像素出发，计算距离图
- 距离 > 1px 的背景像素 = 内部穿孔 → 填为前景
- **不膨胀遮罩，不改变边缘位置**

## 已知限制

1. **色差依赖**：前景和背景颜色必须有一定差异，纯色差法对颜色接近的场景效果差
2. **纹理背景**：木纹、草地等复杂纹理背景容易产生误判
3. **小穿孔**：极端情况下可能有 1-2 个小穿孔残留，可用精修画笔修补
